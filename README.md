# Home-Rental-Platform

A modern home rental platform that helps users discover, explore, and rent properties through an immersive video-first experience. The platform allows property owners to showcase homes using videos, while renters can browse listings, view detailed property information, and connect with owners seamlessly.

---

## Overview

Traditional property listing websites rely heavily on images and text descriptions. Home-Rental-Platform enhances the user experience by incorporating video-based property showcases, allowing users to get a realistic view of homes before making rental decisions.

The platform provides a responsive and modern interface designed to simplify property discovery and rental management.

## Installation

### Clone Repository

```bash
git clone https://github.com/UnnatiVishnoi/Home-Rental-Platform.git
cd Home-Rental-Platform
```

### Install Frontend Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm run dev
```

### Run Backend

```bash
python manage.py runserver
```
